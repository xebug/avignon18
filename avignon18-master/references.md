* Merger \<project-a\> dans \<project-b\>?
  ```bash
  $ cd </path/project-b/>
  $ git remote add project-remote </path/project-a>
  $ git fetch project-remote
  $ git merge --allow-unrelated-histories project-remote/<branch> 
  $ git remote remove project-remote
  ```
